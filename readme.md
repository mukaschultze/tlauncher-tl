# TLauncher - AUR

The best Minecraft launcher.

## Installation

Clone this repo

```bash
git clone --depth=1 git@github.com:mukaschultze/tlauncher-tl.git tlauncher
```

Open the installer folder

```bash
cd tlauncher
```

Start installation

```bash
makepkg -si
```

Go to previous directory

```bash
cd ..
```

Remove temporary installer files

```bash
rm -rf tlauncher
```
